#### MessageBrokerPoc
 - Um projeto bem simples mostrando a integração entre um client e server utilizando RabbitMq com MassTransit.
#### Comando docker para o Rabbitmq
 - docker run -d --hostname rabbitmq --name rabbitmq -e RABBITMQ_DEFAULT_USER=admin -e RABBITMQ_DEFAULT_PASS=admin -p 15672:15672 -p 5672:5672 rabbitmq:management
